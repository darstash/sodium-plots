# Introduction
This repository contains R scripts that clean, analyze, and create figures for the manuscript "Sodium-enriched nectar alters plant-pollinator interactions but does not uniformly attract pollinators" submitted to Ecology. We conducted floral and pollinator surveys in paired control and sodium-addition plots and measured sodium in the soil, leaves, and nectar. 

# Workflow
## Data
Download the data files from Dryad and save them in a subfolder called "data" in the same folder as the scripts.

## Coding scripts
Run the R scripts in the order that they are numbered (see below). Script #2 MUST be run before script #3 to properly create Figure 1.  
1 - pollinator_surveys.R  
2 - plot_sodium_2024.R  
3 - plot_sodium_2025.R  

# Usage
Main analyses were conducted using R version 4.4.1 (2024-06-14) (R Core Team 2021).
